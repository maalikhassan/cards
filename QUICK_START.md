# Quick Start Guide

## Getting Started with LUCK & LOGIC

### Option 1: Direct Use (No Installation)
1. Download the `cards.html` file
2. Open it in any web browser (Chrome, Firefox, Safari, Edge)
3. Press **Ctrl+P** or **Cmd+P** to print
4. Cut and play!

### Option 2: View on GitHub
1. Visit the GitHub repository
2. Click the "cards.html" file
3. Click "Raw" to view in your browser
4. Print directly from there

### Option 3: Local Development
If you want to customize the cards:

**On Windows:**
```powershell
# Clone the repository
git clone https://github.com/yourusername/luck-and-logic.git
cd luck-and-logic

# Open cards.html with your default browser
Invoke-Item cards.html
```

**On Mac/Linux:**
```bash
# Clone the repository
git clone https://github.com/yourusername/luck-and-logic.git
cd luck-and-logic

# Open cards.html with your default browser
open cards.html  # macOS
xdg-open cards.html  # Linux
```

## Printing Tips

### For Best Results:
1. **Use Cardstock** - 200-250 GSM thickness
2. **Minimal Margins** - Set to 0.5 inch or less
3. **Color Printing** - For neon effect (optional)
4. **Letter or A4** - Standard paper size
5. **Landscape** (optional) - For larger cards

### File Size
- HTML file: ~40 KB (single file, easy to share)
- Print size: ~2-3 MB PDF (depends on print settings)

## System Requirements

- **Browser:** Any modern browser (2020+)
- **OS:** Windows, macOS, Linux
- **RAM:** Minimal (< 50 MB)
- **Storage:** ~1 MB for files
- **Internet:** Not required (standalone HTML file)

## Troubleshooting

### Cards look blurry when printing?
- Ensure your browser zoom is at 100%
- Check printer resolution settings
- Update your browser to the latest version

### Colors not printing correctly?
- Switch to color printing if available
- Adjust printer color settings
- Check "Background graphics" in print settings

### Text is cut off on printed cards?
- Reduce margins to 0.5 inch
- Use landscape orientation
- Adjust scale to 95-98% in print dialog

### How do I add custom cards?
- See the README.md file for customization instructions
- Or see CONTRIBUTING.md for detailed steps
